# 📊 Dashboard de Análise de Dados - Power BI

## 🔍 Sobre o projeto
Projeto desenvolvido no Power BI com o objetivo de analisar dados e gerar insights através de visualizações interativas.

## 📊 Funcionalidades
- Tratamento e transformação de dados (Power Query)
- Criação de gráficos e indicadores
- Dashboard interativo para análise

## 🚀 Tecnologias utilizadas
- Power BI
